How to install — Google Workspace / Gmail
💻  Gmail — Desktop (Web)
1. Click inside the Full Signature box above → Ctrl+A to select all → Ctrl+C to copy

2. Open Gmail → click the ⚙ Settings icon → See all settings

3. Stay on the General tab → scroll to Signature → click + Create new → name it JSP Solutions – Full

4. Click into the signature editor → Ctrl+V to paste

5. Under Signature defaults, set For new emails → JSP Solutions – Full

6. Repeat for a second signature named JSP Solutions – Condensed using the condensed block, and set it for On reply/forward

7. Scroll to the bottom → click Save Changes
  Gmail — Mac (Chrome or Safari)
1. Open this page in Chrome or Safari

2. Click inside the Full Signature box → ⌘A → ⌘C

3. Follow the same Desktop steps above — Gmail web is the same on Mac

4. Tip: if formatting looks off after pasting, try ⌘Shift+V to paste with original styling preserved
 
📱  Gmail Mobile (iOS)
1. Open Gmail app → tap your profile picture (top right) → Manage your Google Account

2. Or: tap the ☰ Menu → scroll to Settings → tap your account

3. Tap Signature settings → toggle Mobile Signature on

4. Type your name and title in plain text — Gmail mobile does not support rich HTML signatures

Note: Full HTML signatures (with logo) are only supported in Gmail desktop web. Mobile will display a plain-text fallback.
📱  Gmail Mobile (Android)
1. Open Gmail app → tap ☰ Menu → Settings → tap your account

2. Tap Signature → type your name and title

3. Tap OK to save

Note: Same as iOS — Gmail Android supports plain-text signatures only. The full HTML version displays for all recipients when sent from desktop.
